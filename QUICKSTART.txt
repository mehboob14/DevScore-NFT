## Manual Start (If You Prefer)

### Terminal 1: Backend
```powershell
cd backend
.\venv\Scripts\Activate.ps1
uvicorn main:app --reload --port 8000
```

### Terminal 2: Frontend
```powershell
npm run dev
```

## Troubleshooting

**Backend not starting?**
```powershell
cd backend
.\venv\Scripts\Activate.ps1
pip install -r requirements.txt
```

**Frontend won't start?**
```powershell
npm install
npm run dev
```

**Port in use?**
- Change frontend port in `vite.config.ts`
- Change backend port in command: `uvicorn main:app --port 8001`
